# SeaChain MVP - Enterprise Seafood Traceability Platform

Generated prototype for MVP validation.

## Tech Stack

- **Frontend:** React with Vite with TypeScript
- **Backend:** FastAPI with Python
- **Database:** SQLite

## Quick Start

```bash
# Build and start containers
docker compose up --build

# Or run separately:
docker compose build
docker compose up -d
```

## Access

- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:8000
- **API Docs:** http://localhost:8000/docs

## API Endpoints

- `POST /auth/login` - Authenticate user (fisher/buyer/admin), return JWT token
- `POST /catches` - Fisher submits new catch record with GPS, species, quantity, photos
- `GET /catches` - Buyer retrieves all catches from their supplier network (filtered by date/species/vessel)
- `GET /catches/{id}` - Get detailed catch record with full traceability chain and custody history
- `POST /custody/transfer` - Record custody transfer event between supply chain actors

## Stop

```bash
docker compose down
```
